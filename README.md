# Knowit Ljubljana

Created with [Gatsby](https://www.gatsbyjs.org/).

## Before commit
Run prettier to pretify the code.
```sh
yarn prettier
```

## Install gatsby
```sh
npm install --global gatsby-cli
```

and install dependencies inside this folder:
```sh
yarn
```

And run from your CLI:
```sh
yarn start
```

To build, run:
```sh
yarn build
```

To deploy, run:
```
yarn deploy
```
